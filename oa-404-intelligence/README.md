# Opal & Ash 404 Intelligence Plugin

## Installation
1. Upload the `oa-404-intelligence` folder to `/wp-content/plugins/`.
2. Activate the plugin in WordPress Admin.
3. Configure your message under **Settings > Opal & Ash 404**.

## Usage
Add the following shortcode to your custom 404 page or block:
`[oa_404_content]`

## Developer Customization
**Filters:**
- `oa_404_filter_cta`: Modify the contextual link/message based on the URL.
